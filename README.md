# TELOS NEXUS - Estrutura do site

Site estatico organizado para abrir diretamente pelo arquivo `index.html`.

## Arquivos principais

- `index.html`: pagina principal do site.
- `assets/css/main.css`: estilos extraidos do HTML.
- `assets/js/main.js`: interacoes, menu mobile, icones e efeitos de movimento.
- `assets/vendor/lucide/lucide.es`: biblioteca local de icones Lucide usada pelos atributos `data-lucide`.

## Assets do site

- `assets/images/logo_telosnexus.png`: logotipo usado no topo e no rodape.
- `assets/images/bhaskaraconsult-br-2026-06-27-02_12_34.png`: imagem do portfolio Bhaskara Consult.

## Referencias e materiais de criacao

- `references/design-system/design_system.model.html`: modelo de design system usado como referencia.
- `references/source-builds/`: exports e materiais originais que serviram como referencia durante a criacao do site.
